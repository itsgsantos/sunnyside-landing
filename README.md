## 🚀 Technical Features

* **Complex Grid Layouts:** Implementation of a 2x2 grid system that reorders content seamlessly for mobile devices.
* **Responsive Navigation:** Interactive mobile menu toggle built with Vanilla JavaScript.
* **Optimized Imagery:** Use of responsive image patterns to ensure high-quality visuals without compromising performance.
* **Modern CSS Interactivity:** Smooth hover states, custom transitions, and intentional use of negative space to enhance the "creative" feel.
* **Accessible Semantics:** Proper HTML5 structure ensuring that the marketing message is clear for both users and search engines.

## 🛠️ Tech Stack

- **Structure:** HTML5 (Semantic)
- **Styling:** CSS3 (Flexbox, Grid, Custom Properties)
- **Interactions:** Vanilla JavaScript (ES6)
- **Design System:** Provided by Frontend Mentor

## 🧠 Key Learnings

Developing this project was a significant exercise in **asymmetrical grid layouts** and **component-based CSS**.

**Main Takeaways:**
- Mastering **CSS Grid areas** to create complex, alternating content blocks.
- Implementing a mobile-first navigation system that maintains design integrity.
- Working with diverse image aspect ratios and ensuring they scale properly across all breakpoints.
- Managing color palettes and typography to maintain a cohesive visual identity.

## 💻 Getting Started

### Prerequisites
A modern web browser and a code editor.

### Local Execution
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/itsgsantos/sunnyside-landing.git](https://github.com/itsgsantos/sunnyside-landing.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd sunnyside-landing
    ```
3.  **Run the project:**
    Open `index.html` in your browser.
